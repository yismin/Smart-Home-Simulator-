# Smart Home Automation Simulator

A Java console application that simulates a smart home system with devices, rooms, automation rules, and energy monitoring.

## How to Run

### Prerequisites
- Java JDK 8 or higher installed
- Any Java IDE (IntelliJ IDEA, Eclipse, or VS Code)

### Steps to Run

1. **Clone the repository:**

2. **Open in your IDE:**
   - Open the project folder in IntelliJ IDEA (or your preferred IDE)
   - Wait for the IDE to detect the Java project structure

3. **Run the program:**
   - Navigate to `src/Main.java`
   - Right-click on `Main.java` â†’ Run 'Main.main()'
   - Or click the green play button next to `public static void main`

4. **Interact with the menu:**
   - Follow the on-screen menu to control devices, view status, and test automation rules

### Device IDs for Testing
- `L001` - Living Room Light
- `L002` - Bedroom Light
- `L003` - Kitchen Light
- `T001` - Main Thermostat
- `TV001` - Living Room TV
- `S001` - Living Room Sensor
